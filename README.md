## Build

1. Export slices from `Emacs.sketch` (requires [Sketch](https://www.sketch.com))
2. Run `iconutil -c icns Emacs.iconset -o Emacs.icns` (requires [command line tools](https://developer.apple.com/xcode/resources/))
